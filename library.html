知識庫

這裡不是「文章堆疊」，而是一套財務決策體系：用五層架構把你從基礎保障一路帶到退休現金流與資產傳承。

建議順序： ①安全邊界 → ②現金流 → ③投資決策 → ④退休現金流 → ⑤傳承稅務 每篇固定： 問題 → 例子 → 做法 → 避雷 你的下一步： 先做工具（測驗/清單/計算器）把觀念落地
五層決策系統
第一層｜安全邊界

保障、保額、風險結構

先不翻車：把人生大風險關起來，投資才有意義。

  • 保障的正確順序：意外 / 醫療 / 失能 / 壽險
  • 保額怎麼估：用家庭責任反推（白話版）
  • 常見浪費：重複保障、買貴、保額不足
去工具：保單健檢（上線中） 看學習路徑
第二層｜現金流管理

預備金、負債排序、止血能力

先建立緩衝，再談投資；利率與壓力決定優先順序。

  • 緊急預備金：不同人生階段抓法
  • 卡債/信貸/房貸：先還誰？
  • 「先還債 vs 先投資」的判斷框架
去工具：預備金/負債優先順序
第三層｜投資決策系統（深度支柱）

財報 → 估值 → 進出場 → 總體連動

用可重複、可驗證的方法做決策，而不是靠感覺。

  • 三大報表怎麼看：營收、淨利、EPS 與現金流
  • 關鍵比率：資產負債比、流動/速動比、同業比較
  • 估值：本益比意義、合理價位（巴菲特式思路）
  • 紀律：何時進場、何時停損、持有要看什麼
  • 總體：利率/匯率與美台連動對產業的衝擊
看案例：用一家公司示範（待上線）
第四層｜退休現金流系統

缺口估算、提領策略、壓力測試

退休不是一個數字，是一套收入來源與提領規則。

  • 退休三桶金：生活費 / 醫療照護 / 彈性享受
  • 安全提領觀念：避免「保證報酬」式規劃
  • 年金 / 保險 / 投資：怎麼搭配更穩
去工具：退休缺口估算
第五層｜資產傳承與稅務架構（深度支柱）

法律規則 → 稅務門檻 → 家庭架構

先把資料與規則弄清楚，再談遺囑、受益人、信託等工具。

  • 應繼分 / 特留分：怎麼影響分配與衝突
  • 遺產稅：免稅額、扣除額、不計入項目、級距（整理版）
  • 贈與稅：免稅額與級距（整理版）
  • 保險身故金：稅務與受益人設計的常見誤區
  • 遺囑 vs 保險受益人：本質差異與實務影響
