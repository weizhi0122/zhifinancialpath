三大財報如何串成一套投資決策系統？

三大報表不是三張獨立文件， 它們是一套回答三個核心問題的系統：

  • 這家公司賺不賺錢？（損益表）
  • 這家公司穩不穩定？（資產負債表）
  • 這家公司現金是真是假？（現金流量表）

第一部分：損益表 = 賺錢能力分析

1️⃣ 營收成長率

營收成長率 = (本期營收 − 去年營收) ÷ 去年營收

重點不是單一年，而是「連續 3~5 年是否穩定成長」。

2️⃣ 毛利率 / 淨利率

毛利率 = 毛利 ÷ 營收 淨利率 = 稅後淨利 ÷ 營收

毛利率反映產品競爭力， 淨利率反映整體成本控制能力。

3️⃣ EPS 成長率

EPS 成長率 = (本期 EPS − 去年 EPS) ÷ 去年 EPS

若 EPS 成長但股本膨脹（發行新股），需進一步警惕。

第二部分：資產負債表 = 風險結構分析

1️⃣ 資產負債比

負債比 = 總負債 ÷ 總資產

過高代表槓桿壓力大。 產業不同標準不同，但 70% 以上需注意。

2️⃣ 流動比率

流動比率 = 流動資產 ÷ 流動負債

3️⃣ 速動比率

速動比率 = (流動資產 − 存貨) ÷ 流動負債

速動比率較能反映景氣惡化時的短期償債能力。

4️⃣ ROE（股東權益報酬率）

ROE = 稅後淨利 ÷ 股東權益

長期穩定 15% 以上為優質企業特徵之一。

第三部分：現金流量表 = 獲利品質檢驗

1️⃣ 營運現金流 vs 淨利

如果淨利高，但營運現金流為負， 可能表示應收帳款過高或收入品質有問題。

2️⃣ 自由現金流（FCF）

FCF = 營運現金流 − 資本支出

自由現金流為公司真正可以分配給股東的現金。

第四部分：三表交叉驗證（決策核心）

  • 營收成長 + 現金流同步成長 → 健康成長
  • 營收成長 + 現金流下滑 → 警訊
  • 高 ROE + 高負債 → 槓桿推升型成長
  • EPS 成長 + 發股增資 → 稀釋效應
專業投資不是看一個指標，而是看三張報表「是否一致」。

第五部分：顧問型分析清單

  • 最近五年營收是否穩定成長？
  • 毛利率是否有長期下滑趨勢？
  • ROE 是否穩定高於產業平均？
  • 營運現金流是否長期為正？
  • 負債比是否合理？
