退休決策引擎

制度精準 + 年金比較 + 資產模擬 + 安全邊際壓力測試

Level 1｜制度合法性引擎

自動判斷你是否符合：

  • 老年年金給付資格
  • 老年一次金給付資格
  • 一次請領老年給付（舊制）
98年前是否已有年資？ ├─ 否 → 年資≥15 → 年金 │ 年資<15 → 老年一次金 └─ 是 → 是否符合舊制一次請領條件？ ├─ 是 → 年金 vs 一次請領 比較 └─ 否 → 回一般制度

Level 2｜年金 vs 一次請領比較引擎

系統自動計算：

  • 年金每月領多少
  • 一次領多少
  • 幾歲打平
  • 活多久才回本

Level 3｜退休現金流模擬

整合：

  • 投資總額
  • 退休前累積
  • 股息 / 債息 / 成長率拆解
  • 房租收入
  • 通膨
  • 是否再投入

避免雙重計算，完整名目現金流模擬。

Level 4｜安全邊際模型

自動進行壓力測試：

  • 市場回報 -2%
  • 通膨 +1%
  • 模擬撐多久
  • 最低資產點

整體決策結構

制度合法性 ↓ 制度比較 ↓ 退休現金流模擬 ↓ 制度整合策略比較 ↓ 安全邊際壓力測試
本頁為制度與決策架構說明頁。實際數據請進入工具頁計算。
